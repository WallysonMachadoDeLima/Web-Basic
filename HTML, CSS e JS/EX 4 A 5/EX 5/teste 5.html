Lista de significados

lista ordenada
a lista ordenada é definida por itens de lista contidos pelas tags Li e esses itens devem estar contidos entre as tags e cada item da lista deve ter seu conjunto de tags Li mas a lista toda deve pertencer a um único conjunto para facilitar a memorização, lembre-se - ordered list - lista ordenada lista não ordenada
lista não ordenada
é definida por itens de lista contídos pelas tags | e esses itens devem estar contidos entre as tags cada item da lista deve ter seu conjunto de tags Li ma a lista toda deve pertencer a um único conjunto para facilitar a memorização, lembre-se - unordered list - lista não ordenada
lista de definição
a lista de definição é definida por dois modelos de itens, o termo e sua definição. Por Isso, cada item dessa lista é um grupo, composto pelas tags dt e dd. Depois de completar a lista, esses grupos de elementos devem estar contidos em um único conjunto de L. para facilitar a memorização, lembre-se de L de Finish last lista de definição o elemento que funciona como o verbete é o DT definition termo da definição e o elemento que funciona como o significado é o DDD definição description descrição da definição.
